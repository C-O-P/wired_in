Secret Code

In WIRED IN! the core team of PCCUBE decided to give a secret code to each team. The secret code was generated as follows:

  • Lowercase letters were replaced by Uppercase letters.
  • Uppercase letters were replaced by Lowercase letters.
  • Blank space was printed at the place of vowels.

As there are too many teams your task is to help them in generating the secret code.

Input

A string s which don't have more than 105 character.

Output

Print the generated secret code.

Ex:

Input

ReDoc

Output

r d C


Input

aVdtHf1G

Output

vDThF1g
